## 下载&使用
请前往[Releases](https://github.com/obentnet/MuseBot/releases)下载发行版

下载后,使用CMD或Shell，cd进入程序目录下,输入`muse.py`启动程序

## Bot功能

### 上层应用

* AI女友: 可对话交流的AI女朋友
* IP定位: 使用IP定位到当前市级别的位置,附带经纬度
* PING: Ping命令,发包测试.
* cmd: CMD DOS系统(基于Windows)
* baidu: 百度搜索引擎
* news: 最新资讯

### 底层设置
* upgrade: 检查更新
* reboot: 重启Bot
* shutdown: 关闭Bot
* about-muse: 关于MuseBot
